# Host Training Materials
## Shortlet Platform Pilot Program

### Module 1: Platform Basics (30 minutes)

**Topics Covered:**
1. Dashboard navigation
2. Adding and editing properties
3. Managing calendar and availability
4. Understanding pricing rules
5. Payment and payout system

**Video Tutorial:** [Link to video]

**Practice Exercise:**
- Add a test property
- Block dates on calendar
- Update pricing
- Review payout settings

---

### Module 2: Guest Communication (45 minutes)

**Topics Covered:**
1. WhatsApp integration overview
2. Automated messages (booking, check-in, review)
3. Responding to guest inquiries
4. Handling special requests
5. Resolving guest issues

**Best Practices:**
- Response time targets (<2 hours)
- Professional tone examples
- Common scenarios and responses
- Escalation procedures

**Templates Provided:**
- Welcome message
- Check-in instructions
- House rules reminder
- Checkout instructions
- Review request

---

### Module 3: Maximizing Revenue (60 minutes)

**Topics Covered:**
1. Competitive pricing strategies
2. Dynamic pricing (weekends, seasons)
3. Length-of-stay discounts
4. Last-minute pricing
5. Occupancy optimization

**Case Studies:**
- Host A: Increased revenue 40% with dynamic pricing
- Host B: 80% occupancy with competitive rates
- Host C: Premium pricing with exceptional service

**Tools:**
- Pricing calculator
- Market rate comparison
- Revenue forecasting

---

### Module 4: Property Photography (30 minutes)

**Topics Covered:**
1. Importance of professional photos
2. What to photograph
3. Lighting and composition
4. Photo editing basics
5. Ordering sequence

**Photo Checklist:**
- [ ] Exterior/building entrance
- [ ] Living room (2-3 angles)
- [ ] Each bedroom
- [ ] Kitchen
- [ ] All bathrooms
- [ ] Balcony/outdoor space
- [ ] Amenities (pool, gym, etc.)
- [ ] Parking area
- [ ] Building facilities

**Free Photography:**
- Pilot hosts get 1 free professional session
- Book via dashboard
- Photographer visits within 3 days
- Photos delivered in 48 hours

---

### Module 5: Guest Experience (45 minutes)

**Topics Covered:**
1. Creating memorable stays
2. Property preparation checklist
3. Welcome amenities
4. Handling complaints
5. Getting 5-star reviews

**Property Preparation:**
- Deep cleaning checklist
- Amenity stock levels
- Maintenance checks
- Welcome pack ideas

**5-Star Service:**
- Exceed expectations
- Personal touches
- Quick problem resolution
- Follow-up after checkout

---

### Module 6: Channel Manager (30 minutes)

**Topics Covered:**
1. What is a channel manager?
2. Connecting Airbnb
3. Connecting Booking.com
4. Calendar synchronization
5. Avoiding double bookings

**Step-by-Step:**
1. Get iCal URL from Airbnb
2. Paste in dashboard settings
3. Verify sync working
4. Test with manual block
5. Monitor for conflicts

---

### Module 7: Analytics & Growth (30 minutes)

**Topics Covered:**
1. Understanding your dashboard metrics
2. Occupancy rate optimization
3. Revenue per available night (RevPAN)
4. Guest acquisition cost
5. Repeat guest rate

**Key Metrics:**
- Occupancy rate (target: 60%+)
- Average nightly rate
- Revenue per month
- Response rate (target: 90%+)
- Guest rating (target: 4.5+)

**Growth Strategies:**
- Seasonal adjustments
- Marketing tactics
- Referral programs
- Repeat guest incentives

---

### Module 8: Legal & Compliance (20 minutes)

**Topics Covered:**
1. Lagos shortlet regulations
2. Tax obligations
3. Guest registration requirements
4. Insurance recommendations
5. Liability protection

**Required Documents:**
- Property ownership or lease agreement
- Valid ID
- Tax identification number (TIN)
- Business registration (optional)
