# Pilot Recruitment Campaign Launch Checklist

## Pre-Launch (1 week before)

### Content Preparation
- [ ] Finalize all social media posts
- [ ] Create graphics for LinkedIn, Instagram, Facebook
- [ ] Record video testimonial (if available)
- [ ] Prepare FAQ document
- [ ] Set up Google Forms application
- [ ] Configure email automation
- [ ] Create applicant tracking spreadsheet

### Technical Setup
- [ ] Set up Google Forms with all questions
- [ ] Configure form notifications to pilot@yourplatform.com
- [ ] Test form submission and email automation
- [ ] Set up WhatsApp Business account
- [ ] Create WhatsApp broadcast list
- [ ] Set up email forwarding rules
- [ ] Create calendar for interviews

### Team Preparation
- [ ] Brief team on pilot program details
- [ ] Assign roles (screener, interviewer, coordinator)
- [ ] Schedule team availability for interviews
- [ ] Prepare interview script and evaluation rubric
- [ ] Set up Zoom/Google Meet for interviews

## Launch Day (Day 1)

### Morning (9 AM WAT)
- [ ] Post LinkedIn recruitment post
- [ ] Share on Twitter/X (thread)
- [ ] Post on Instagram with property images
- [ ] Post on Facebook with detailed description
- [ ] Send WhatsApp status update
- [ ] Email existing contacts/newsletter

### Afternoon (2 PM WAT)
- [ ] Monitor application submissions
- [ ] Respond to questions on social media
- [ ] Engage with comments and shares
- [ ] Send confirmation emails to applicants

### Evening (6 PM WAT)
- [ ] Review day 1 metrics (applications, engagement)
- [ ] Respond to all pending questions
- [ ] Plan day 2 activities

## Days 2-7 (Application Period)

### Daily Tasks
- [ ] Post reminder on at least 2 platforms
- [ ] Respond to all questions within 2 hours
- [ ] Send confirmation emails to new applicants
- [ ] Update applicant tracking spreadsheet
- [ ] Share user-generated content (if any)

### Mid-Week (Day 4)
- [ ] Post progress update ("X hosts applied!")
- [ ] Share testimonial or success story
- [ ] Boost top-performing posts (if budget allows)
- [ ] Reach out to potential hosts directly

### End of Week (Day 7)
- [ ] Post "Last chance to apply!" reminder
- [ ] Send email to newsletter subscribers
- [ ] Final push on all social media channels

## Post-Application Period (Days 8-14)

### Week 2: Screening & Interviews
- [ ] Score all applications using selection criteria
- [ ] Shortlist top 15 candidates
- [ ] Send interview invitations to shortlisted candidates
- [ ] Schedule 30-minute interviews
- [ ] Conduct interviews with evaluation rubric
- [ ] Send rejection emails to non-shortlisted applicants

### Week 2: Final Selection
- [ ] Review interview scores
- [ ] Select 10 hosts (8 primary + 2 backup)
- [ ] Send acceptance emails to selected hosts
- [ ] Send rejection emails to interviewed but not selected
- [ ] Confirm onboarding session attendance

## Post-Selection (Weeks 3-4)

### Onboarding Preparation
- [ ] Send welcome package to selected hosts
- [ ] Create onboarding presentation
- [ ] Prepare training materials
- [ ] Set up host accounts on platform
- [ ] Schedule photography sessions
- [ ] Create WhatsApp group for pilot hosts

### Onboarding Session
- [ ] Conduct 2-hour onboarding (online or in-person)
- [ ] Provide platform walkthrough
- [ ] Answer questions and concerns
- [ ] Collect feedback on onboarding process

### Launch Support
- [ ] Help hosts list their first property
- [ ] Verify Airbnb calendar sync
- [ ] Test payment integration
- [ ] Monitor first bookings
- [ ] Provide ongoing support

## Metrics to Track

### Application Metrics
- Total applications received
- Applications by source (LinkedIn, Instagram, etc.)
- Applications by location (Lekki, VI, Ikoyi)
- Average application score
- Conversion rate (applicants → selected)

### Engagement Metrics
- Social media reach and impressions
- Post engagement rate (likes, comments, shares)
- Click-through rate to application form
- Email open and response rates
- WhatsApp broadcast engagement

### Quality Metrics
- Average host experience (years)
- Average properties per host
- Average monthly bookings
- Diversity of property types
- Platform presence (multi-channel hosts)

## Success Criteria

✅ 50+ applications received
✅ 15+ qualified candidates (score ≥ 60/100)
✅ 10 hosts selected and onboarded
✅ 8+ hosts list property within week 1
✅ 5+ bookings completed in month 1
✅ 80%+ host satisfaction (survey)
